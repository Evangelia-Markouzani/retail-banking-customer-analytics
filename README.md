# Retail Banking Customer Analytics

Retail Banking Customer Segmentation, Cross-Sell Strategy & Customer Scoring Analysis  
using SQL, Python & Tableau.

---

## Project Overview

This project analyzes retail banking customer data in order to:

- Identify customer segments
- Evaluate investment product adoption
- Detect cross-selling opportunities
- Prioritize high-value customers using a scoring model

The analysis uses SQL for data extraction, Python for exploration and modeling,  
and Tableau for interactive dashboard visualization.

---

## Business Objective

Retail banks aim to understand customer behavior to improve product adoption  
and increase customer value. This project provides insights on:

- Customer segmentation by income & balance
- Correlation between key metrics
- Strategic targeting for cross-selling initiatives
